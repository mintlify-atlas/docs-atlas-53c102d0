ML Intern is your autonomous ML engineering assistant. Give it a task in plain language — "fine-tune Llama on my dataset", "find the best training recipe for image classification", "benchmark this model on GLUE" — and it researches the literature, writes the code, spins up cloud compute on Hugging Face Jobs, and pushes the results to the Hub.

## Get started in three steps

<Steps>
  <Step title="Install ML Intern">
    Clone the repo and install with `uv`:

    ```bash
    git clone git@github.com:huggingface/ml-intern.git
    cd ml-intern
    uv sync
    uv tool install -e .
    ```
  </Step>
  <Step title="Set your credentials">
    Create a `.env` file in the project root with your API keys:

    ```bash .env
    HF_TOKEN=<your-hugging-face-token>
    ANTHROPIC_API_KEY=<your-anthropic-api-key>
    GITHUB_TOKEN=<your-github-personal-access-token>
    ```

    If `HF_TOKEN` is not set, the CLI will prompt you to paste one on first launch.
  </Step>
  <Step title="Start your first task">
    Launch the interactive chat or pass a prompt directly:

    ```bash
    # Interactive mode
    ml-intern

    # Headless mode — runs the task and exits
    ml-intern "fine-tune llama on my dataset"
    ```
  </Step>
</Steps>

<Note>
  ML Intern uses Claude (via Anthropic or AWS Bedrock) as its default model. You need either an `ANTHROPIC_API_KEY` or a configured Bedrock profile to use it. You can switch models at runtime with the `/model` slash command.
</Note>
